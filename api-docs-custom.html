Keyboard Shortcuts & Help

Keyboard Shortcuts

  • Ctrl + Enter - Send request
  • Ctrl + K - Focus URL input
  • Ctrl + H - Toggle history panel
  • Ctrl + F - Toggle favorites panel
  • Ctrl + D - Toggle dark mode
  • Ctrl + N - New request
  • Esc - Close modal

Features

  • Click any endpoint in the documentation to auto-fill the request
  • Save frequently used requests as favorites
  • View your request history and replay any request
  • Configure multiple auth profiles for different environments
  • Import/export favorites for backup or sharing
  • Dark mode support for comfortable viewing

Auth Profiles

Configure authentication profiles in the settings. Each profile can have its own base URL and access token. The token will be automatically included in all requests.

🔗 Redoc Integration

  • Click endpoint in docs: Click any endpoint in Redoc (left panel) to auto-populate the request builder
  • 🚀 Try in Panel button: Quick-load buttons appear next to each endpoint for fast access
  • Visual feedback: Selected endpoint highlights with colored background for 2 seconds
  • Example bodies: Request body examples are auto-generated from OpenAPI schemas
  • Query parameters: Parameters from the OpenAPI spec are automatically filled in
